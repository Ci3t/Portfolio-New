## Visual Direction

- Treat this as an "operator profile" dossier.
- Use a split grid:
  - left: animated profile/bio block
  - right: live capability readout panels
- Avoid a plain paragraph-only about section.
- Use numbered rows: `01`, `02`, `03`, `04`.

## Animation Ideas

- Bio lines reveal with staggered masks.
- Capability panels enter like system modules being mounted.
- On hover, each strength card reveals a second layer with proof or stack tags.
- Scroll-linked horizontal scan line moves through the section.

## HR/Dev Clarity

Use simple labels with creative secondary labels. Example: `UI Engineering` can have `INTERFACE LAYER` as metadata, not as the only readable title.
