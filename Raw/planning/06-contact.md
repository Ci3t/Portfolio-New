## Visual Direction

- Keep the "Secure Uplink" terminal concept from the Stitch design.
- Use a terminal-inspired contact panel, but do not make users type commands to contact.
- Include simple, obvious buttons:
  - Email
  - LinkedIn
  - GitHub
  - Download CV

## Animation Ideas

- Terminal panel boots as it enters viewport.
- Contact methods reveal as system checks:
  - `EMAIL: READY`
  - `GITHUB: ONLINE`
  - `LINKEDIN: ONLINE`
  - `CV: AVAILABLE`
- CTA hover can flicker once, then settle.
- Copy email button can show `COPIED` status.

## HR/Dev Clarity

Do not hide contact behind a gimmick. The final section can be stylized, but the action buttons must be instantly understandable.

Use the downloadable CV later once the CV is refreshed or approved. Until then, the contact section can include a disabled/planned CV state or link to the existing CV only if Rani approves.
